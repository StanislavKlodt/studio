Wellness-приложение Fitability

  • МОБИЛЬНЫЕ ПРИЛОЖЕНИЯ

Fitability — приложение для тренировок, разработанных по авторской программе.

Требования

Принимая во внимание тематику контента, который заложен в основу приложения, его целевой аудиторией являются люди, которые хотят улучшить свою физическую форму и нуждаются в программе, по которой можно это сделать пошагово. Таким образом, приложение направлено на предоставление информации пользователям с разным уровнем физической подготовки вне зависимости от места тренировок. Контент был подготовлен тренером с многолетним опытом работы в сфере спорта.

Анализ и предложение

Основой для разработки сервиса стало понимание специфики фитнес-программ и желание персонализировать контент под актуальные потребности и состояние конечного пользователя. Решено было ориентироваться на данные, введенные пользователем, такие как: возраст, вес, физическое состояние и желаемые результаты. Алгоритм сопоставляет их с программами, посредством таксономии и классификации, для предоставления релевантного контента.

Функциональность и дизайн

Для того, чтобы пользователи могли приступить к использованию приложения как можно быстрее, им достаточно пройти опрос из пары этапов, чтобы предоставить сервису данные, которые будут использованы для создания исходного профиля и составления личного плана тренировок.

Приложение оформлено в минималистичном стиле, чтобы сосредоточить пользователя на контенте.

После ввода личных система собирает персонализированную программу тренировок, которую пользователь может использовать в своих повседневных тренировках. Обновление личных данных приводит к смене контента, поэтому приложение разработано так, чтобы адаптироваться и расти вместе с пользователем.

Один из способов привлечение новых пользователей, реализован за счет того, что пользователь может поделиться своим прогрессом в популярных социальных сетях, побуждая других скачать приложение. Также была введена модель «freemium», согласно которой новый пользователь может подписаться на пробный период с 10-минутными тренировками. Для снятия ограничения нужно оформить платную подписку.

Заключение

Приложение стало коммерчески успешным проектом для нашего заказчика. Активное распространение пользователями своих результатов и обновление контента автором программ не только создали новый источник дохода для бизнеса, но и увеличили аудиторию личного бренда. Не менее важно, продукт помог многим людям стать здоровее и счастливее. Подобное модель разработки приложение подойдет для проектов, которые получают трафик из социальных сетей и хотят создать дополнительный источник монетизации своей экспертизы.
